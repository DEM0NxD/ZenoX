PRIVACY
POLICY

We respect your privacy and are committed to protecting it. Here's everything you need to know about how ZenoX handles your data.

Information We Collect

We only collect the minimum data required for ZenoX to function correctly. This may include:

  • User IDs, server IDs, channel IDs, and message IDs necessary for bot operations
  • Configuration data created through bot commands (e.g., moderation settings, automod rules)
  • Audit logs and moderation records as configured by server administrators

We do not collect personal information such as real names, email addresses, passwords, payment details, or any other sensitive personal data.

How We Use Information

Collected data is used strictly to provide ZenoX's core features. This includes:

  • Running moderation, antinuke, and automation systems
  • Storing server settings and configuration preferences
  • Processing commands and system functionality in real-time
  • Generating audit logs at the server admin's request

Data is processed automatically and is never reviewed manually unless required for debugging, abuse prevention, or critical security purposes.

Data Storage & Security

All data is stored securely within private services and controlled environments. We take reasonable technical and organizational measures to prevent unauthorized access, misuse, or alteration of stored data.

  • Access to stored data is restricted to authorized systems only
  • Infrastructure is monitored continuously for unusual activity
  • Security practices are reviewed and updated regularly
No Selling or Sharing

We do not sell, resell, trade, or rent any collected data under any circumstances. Data is never shared with third parties, advertisers, or external services, except when:

  • Required by applicable law or legal process
  • Necessary to prevent abuse, fraud, or active security threats
  • Explicitly authorized by the server owner or data subject
Data Retention & Deletion

Data is retained only as long as it is necessary for the service to operate correctly. You have options to manage your data:

  • Server owners may request data removal by removing ZenoX or resetting configurations via commands
  • Once removed, associated data may be permanently deleted from our systems
  • Individual users may contact us to request deletion of any stored records associated with their user ID
Service Availability & Data Loss

While we strive for 24/7 reliability and uptime, we are not responsible for data loss caused by:

  • Scheduled maintenance or unplanned outages
  • System crashes or infrastructure failures
  • Force-majeure events outside our control

Backups may not always be available or complete. We recommend server administrators maintain their own configuration records for critical settings.

Changes to This Policy

This Privacy Policy may be updated from time to time to reflect improvements, legal requirements, or security changes. When updates occur:

  • The "Last updated" date at the top of this page will be revised
  • Significant changes may be announced in our support server
  • Continued use of ZenoX after updates constitutes acceptance of the revised policy
Contact Us

If you have questions about this Privacy Policy or wish to request data deletion, you can reach us through:

  • Our official Discord support server — the fastest way to get a response
  • Opening a support ticket via the ZenoX ticket system in our server
Support Server
